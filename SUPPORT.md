# Support

This is a personal open-source project maintained in spare time. Response times are best-effort.

## Where to get help

| Channel | Use for |
|---|---|
| [GitHub Issues](../../issues) | Bug reports and feature requests — use the templates |
| [GitHub Discussions](../../discussions) | Questions and general conversation |
| Repository `README.md` | Setup, usage, and configuration |

## Before opening an issue

- Search existing [issues](../../issues) and [discussions](../../discussions).
- Check that you are on the **latest release**.
- Read the `README.md` and any linked docs.

## Security issues

Use **private disclosure** — see [SECURITY.md](SECURITY.md). Do not open a public issue.
